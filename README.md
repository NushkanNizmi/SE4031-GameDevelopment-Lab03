# Lab 03 – Environment Building + Materials + Lighting + VR Scale

Unity Version: 2022.3 LTS  
Lab Duration: 2 Hours  

---

## Objective
To build a VR-safe environment by applying correct real-world scale, creating and applying materials, and configuring lighting for comfortable VR navigation.

---

## Learning Outcomes
After completing this lab, students will be able to:

- Apply real-world scale rules in VR environments
- Create and manage materials in Unity
- Apply materials correctly to floors, walls, and props
- Configure directional and point lighting suitable for VR
- Create a themed environment with at least 8 props
- Validate the environment through VR navigation

---

## Prerequisites
This lab continues the same project from Lab 02:
- Scene: Assets/_Project/Scenes/Main.unity
- Locomotion must work (teleport or smooth move)
